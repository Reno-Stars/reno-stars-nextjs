# 2026-03-11 — Add "Whole House" Tab to Admin Sites List

## Summary

Added a third tab to the admin sites list page that filters to show only whole-house sites (`showAsProject === true`). Extracted shared `siteActions` callback to eliminate duplication between tabs.

## Changes

### Modified Files

- **`app/admin/(dashboard)/sites/SitesListClient.tsx`** — Added `'wholehouse'` to tab state union type. Added `wholeHouseSites` memo filtering sites by `showAsProject`. Added third tab button with count badge and ARIA attributes. Added tab panel reusing existing `columns`, `filterRow`, `renderExpandedRow`, and new shared `siteActions` callback. Extracted `siteActions` as `useCallback` to deduplicate the actions column render between "All Sites" and "Whole House" tabs.
- **`messages/admin/en.ts`** — Added `tabWholeHouse: 'Whole House'` translation key.
- **`messages/admin/zh.ts`** — Added `tabWholeHouse: '全屋装修'` translation key.

### Documentation

- **`docs/architecture.md`** — Updated Sites admin list description from "Two-tab UI" to "Three-tab UI" with Whole House tab details.
