# Edit History: 2026-02-27

## Summary

**Session 1** — Independent site budget and duration fields. Previously, site budget and duration were aggregated (summed) from child projects via `calculateCombinedBudget()` and `aggregateDurations()`. Changed to independent fields stored directly on the `project_sites` table. Removed 3 dead helper functions and ~130 lines of tests. Updated AI generators (site description + blog generation) to include budget/duration. Updated public translation labels from "Combined Budget"/"Total Timeline" to "Budget"/"Timeline".

### Changes Made

1. **Schema**: Added `budgetRange` (varchar 50), `durationEn` (varchar 100), `durationZh` (varchar 100) columns to `project_sites` table in `lib/db/schema.ts`.
2. **Types**: Added `budget_range` and `duration` (Localized) to `Site` and `LocalizedSite` in `lib/types.ts`. Removed `totalBudget` and `totalDuration` from `SiteAggregated` and `LocalizedSiteAggregated`.
3. **Query layer**: Updated `mapDbSiteToSite` to map new fields. Removed calls to `calculateCombinedBudget()` and `aggregateDurations()` from `getSiteBySlugFromDb` and `getSitesAsProjectsFromDb`.
4. **Dead code removal**: Deleted `parseBudgetRange()`, `calculateCombinedBudget()`, `aggregateDurations()` from `lib/db/helpers.ts` (~70 lines) and corresponding tests (~130 lines) from `tests/unit/db/helpers.test.ts`.
5. **Localization helpers**: Updated `getLocalizedSite` and `getLocalizedSiteWithProjects` in `lib/data/projects.ts` to use site's own budget/duration instead of aggregated values.
6. **Admin form**: Added controlled state inputs (`useState`) for budget/duration in `SiteForm.tsx`. Updated `handleAIGenerate` callback to populate these fields from AI response.
7. **Server actions**: Updated `getSiteData()` in `app/actions/admin/sites.ts` to extract budget/duration from FormData. Updated admin detail page to pass fields to form.
8. **Display components**: `SiteDetailPage` and `ProjectsPage` now use `site.budget_range` and `site.duration` instead of aggregated values.
9. **AI site generator**: Added `budgetRange`, `durationEn`, `durationZh` to `SiteDescriptionSchema` and `SITE_DESCRIPTION_PROMPT` in `lib/ai/content-optimizer.ts`.
10. **AI blog generator**: Added fields to `SiteDataForBlog` interface and context builder in `lib/ai/blog-generator.ts`. Updated `generateBlogFromSite` action to pass fields.
11. **Translations**: Added `sites.budgetRange`, `sites.duration`, and their tooltips to admin translations (EN + ZH). Updated public translations: "Combined Budget" → "Budget", "Total Timeline" → "Timeline".
