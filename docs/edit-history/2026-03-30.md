# 2026-03-30 ‚Äî Self-Hosted Image Optimization

## Summary

Added self-hosted image optimization using `sharp` to fix critical performance issues: 13.3s LCP, 56/100 mobile PageSpeed score. Bypasses Vercel's image optimization quota (`unoptimized: true` stays in `next.config.ts`). All homepage images now served as resized WebP via `/api/image` endpoint.

## New Files

### `app/api/image/route.ts`
- Image optimization API route using `sharp` (Node.js runtime)
- Accepts `url`, `w` (width), `q` (quality), `f` (format: webp/avif)
- URL allowlist: reno-stars.com, R2 buckets, googleusercontent.com, localhost (dev)
- SSRF protection: `redirect: 'follow'` with post-redirect hostname validation, host allowlist
- Size guard: 50MB limit via Content-Length header + post-download byteLength check
- Snaps width to shared `SRCSET_WIDTHS` breakpoints
- Cache-Control: immutable (1 year) for CDN/browser caching
- In-memory LRU cache (50 entries, 1hr TTL) for long-lived processes
- Error fallback: returns 502 JSON (no open redirect)

### `lib/image.ts`
- Shared image optimization constants and URL builders
- `SRCSET_WIDTHS`: [320, 640, 828, 1080, 1200, 1920]
- `buildOptimizedUrl()`, `buildSrcSet()`, `buildPreloadUrl()`
- Used by both `OptimizedImage` component and API route

### `components/OptimizedImage.tsx`
- Client component, drop-in replacement for `next/image`
- Generates responsive `srcSet` at 6 breakpoints via `/api/image`
- `onError` fallback to original `src` (with `data-fallback` loop guard)
- `fetchPriority="high"` when `priority` prop is set
- Local paths (e.g. `/worksafe-bc-logo.jpg`) bypass optimization
- Supports `fill` mode (absolute positioning + object-fit: cover)

**Root cause:** `ProjectsPage.tsx` creates `DisplayProject` objects for site projects (identified by "Whole House" badge). The site‚ÜíDisplayProject mapping omitted `hero_video`, so even though the DB had the video URL and the query layer mapped it correctly, the field was lost during conversion.

- Added `hero_video: site.hero_video` to the site‚ÜíDisplayProject mapping
