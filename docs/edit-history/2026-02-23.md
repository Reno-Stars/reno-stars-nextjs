# Edit History: 2026-02-23

## Summary

**Session 7** — Admin UX audit and fixes. A comprehensive PM-level UX review of the admin panel identified 8 issues across bilingual consistency, workflow improvements, and visual polish. All 8 were implemented and verified.

### Changes Made

1. **Bilingual error/loading pages**: `error.tsx` and `loading.tsx` now use `useAdminTranslations()` instead of hardcoded English. Loading page redesigned with animated CSS spinner.
2. **ToggleButton bilingual defaults**: Removed hardcoded `'Yes'`/`'No'` defaults; now uses `t.common.yes`/`t.common.no` from admin translations for proper EN/ZH switching.
3. **Contact status label translations**: Filter buttons and status dropdown in `ContactsListClient` now use `t.status[s]` instead of raw English enum values (e.g., `'new'` → `'New'`/`'新的'`).
4. **Back navigation on edit/new pages**: Added `backHref` and `backLabelKey` props to `AdminPageHeader`. Applied to 14 edit/new page files across blog, FAQs, gallery, services, partners, service areas, trust badges, and social links.
5. **Standardized toggle buttons (blog + sites)**: Replaced raw `<button>` publish toggles in `BlogListClient` and `SitesListClient` with the shared `ToggleButton` component, adding `pendingId` state for loading feedback.
6. **Dashboard pre-filter for new contacts**: Changed "New Contacts" card href from `/admin/contacts` to `/admin/contacts?status=new`. Contacts page reads `?status` query param via `useSearchParams` to set the initial filter.
7. **Delete button color standardization**: Changed delete buttons from gray (`TEXT_MID`) to red (`ERROR`) across all 6 list pages: gallery, service areas, trust badges, FAQs, services, and partners.
8. **Loading spinner**: Added animated gold CSS spinner to admin loading page (done as part of fix #1).

## Session 8 — Admin UX Audit Round 2

### Summary

PM-level UX audit identified 18 issues in 3 priority tiers. All 12 actionable items implemented, plus 6 code review fixes from two subsequent review passes. Total: 18 files changed, 338 insertions, 210 deletions.

### Changes Made

#### UX Fixes (12 items)

1. **Social links "Add New" button + delete support**: Added `actionKey`/`actionHref` to social links `AdminPageHeader`. Created `deleteSocialLink` server action. Added `ConfirmDialog` + delete button to `SocialLinksListClient`.
2. **DataTable search-aware empty state**: Shows `"No results for '{query}'"` when search is active, generic `"No records found."` otherwise.
3. **BilingualInput maxLength hint**: Shows bilingual `(max N)` next to label via `t.common.maxLength` translation key.
4. **Service area slug tooltip**: Read-only slug field shows tooltip explaining slugs cannot be changed after creation.
5. **Standardized action button gaps**: Changed from `0.75rem` to `0.5rem` with `justifyContent: 'flex-end'` across FAQs, Services, Service Areas, Trust Badges list pages.
6. **ProjectForm EditModeToggle**: Replaced 35-line inline edit/cancel button block with shared `EditModeToggle` component.
7. **SiteForm EditModeToggle**: Same standardization as ProjectForm.
8. **ProjectForm remove button styling**: Added `border: 1px solid ERROR`, `borderRadius: 4px`, `padding: 0.25rem 0.5rem` to scope and external product remove buttons.
9. **ImagePairEditor upload spinner**: Gold spinning border animation using global `@keyframes admin-spin`.
10. **CompanyForm visual grouping**: Grouped 12 fields into 4 labeled sections (Business Info, Location, Legal, Marketing) with `SURFACE` background containers.
11. **TopBar logout confirmation**: Replaced direct `<form action={logoutAction}>` with `ConfirmDialog` (`variant="warning"`).
12. **ConfirmDialog focus trap**: Tab/Shift+Tab cycles within the dialog's focusable elements.

#### Code Review Fixes (6 items from 2 review passes)

1. **ImagePairEditor**: Replaced inline `@keyframes spin` with global `admin-spin` to avoid DOM injection and naming conflicts.
2. **SocialLinksListClient**: Added missing `padding: 0` on delete button for consistency.
3. **CompanyForm**: Extracted repeated section header style to `sectionHeaderStyle` constant; added `aria-hidden="true"` on grid spacer divs.
4. **BilingualInput**: Internationalized `(max N)` hint from hardcoded English to `t.common.maxLength`.
5. **TopBar**: Added `setShowLogout(false)` before logout action for robustness.
6. **Unused imports cleanup**: Removed `NAVY` from CompanyForm, `GOLD`/`TEXT_MID` from SiteForm.
