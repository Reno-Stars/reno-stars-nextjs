## Google Reviews Migration for Structured Data

Replaced hardcoded `rating`, `reviewCount`, and `ratingSource` fields in the Company model with live Google Places API data for all JSON-LD structured data schemas.

### Removed Fields

| Location | Change |
|----------|--------|
| `lib/types.ts` | Removed `rating`, `reviewCount`, `ratingSource` from `Company` interface |
| `lib/db/schema.ts` | Removed columns from `companyInfo` table |
| `lib/db/queries.ts` | Removed mapping from `getCompanyFromDb()` |

| `app/admin/(dashboard)/company/CompanyForm.tsx` | Removed 3 editable input fields |
| `app/actions/admin/company.ts` | Removed from update action |

### Updated Structured Data Schemas

All three schemas now accept optional `googleRating?: number` and `googleReviewCount?: number` props. Ratings use conditional spread to only emit `aggregateRating` when Google data is available:

| File | Before | After |
|------|--------|-------|
| `LocalBusinessSchema.tsx` | Parsed `company.rating` ("10/10" format), `parseRating()` | Google Reviews props, `bestRating: 5` |
| `ProjectSchema.tsx` | `company.rating.split('/')` | Google Reviews props |
| `LocalBusinessAreaSchema.tsx` | `company.rating.match()` regex | Google Reviews props |
| `ReviewSchema.tsx` | Emitted duplicate `aggregateRating` | Removed (layout handles it globally) |

### Data Flow

Pages fetch `getGoogleReviews()` alongside existing data in `Promise.all`:

| Page | Props Passed To |
|------|-----------------|
| `app/[locale]/layout.tsx` | `LocalBusinessSchema` |
| `app/[locale]/projects/[slug]/page.tsx` | `ProjectSchema` |
| `app/[locale]/areas/[city]/page.tsx` | `LocalBusinessAreaSchema` |

## Logo Preview Fallback

**`components/admin/ImageUrlInput.tsx`** — Fixed missing logo preview in CompanyForm. When `getAssetUrl()` rewrites a production URL to MinIO and the file doesn't exist locally, the `onError` handler now falls back to the original URL before giving up:

```typescript
onError={(e) => {
  const img = e.currentTarget;
  if (img.src !== url && url.startsWith('http')) {
    img.src = url;  // Try original URL before MinIO rewrite
  } else {
    setImageError(true);
  }
}}
```
