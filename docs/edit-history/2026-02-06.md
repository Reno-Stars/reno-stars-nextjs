## Code Review Fixes

### Performance Optimizations

1. **Parallel database updates** (`app/actions/admin/projects.ts`)
   - Changed sequential `for` loop to `Promise.all()` for reordering projects
   - Reduces N round-trips to 1 concurrent batch

2. **React hooks optimization** (`components/admin/HouseStack.tsx`)
   - Added `useMemo` for sorted projects
   - Added `useCallback` for all event handlers
   - Static styles moved outside component

### Accessibility Improvements

1. **Keyboard reordering** (`components/admin/HouseStack.tsx`)
   - Alt+Up/Down for keyboard-based reordering
   - Enter key to select items
   - `aria-live="polite"` for project count announcements

2. **Screen reader support**
   - Proper `aria-label` attributes on interactive elements
   - Translation-based labels for dynamic content

### Code Quality

1. **Reusable Tooltip component**
   - Extracted from 5+ files with duplicated tooltip logic
   - Supports `size` prop ('sm' for checkboxes, 'md' for form fields)

2. **Shared constants**
   - `SERVICE_TYPES` constant with TypeScript union type
   - `SPACE_TYPES` with EN/ZH mappings
   - `SPACE_TYPE_TO_ZH` and `SERVICE_TYPE_TO_CATEGORY` lookup maps

3. **Generic toggle pattern** (`app/actions/admin/sites.ts`)
   - Single `toggleSiteField()` helper for all boolean toggles
   - Reduces code duplication from 3 nearly-identical functions

4. **Batch upload error handling** (`components/admin/ProjectForm.tsx`)
   - Continues processing all files even if some fail
   - Collects errors and shows summary message
